# 문제
- 점수 N을 입력받은 뒤, 주어진 코드의 주석부분을 switch문을 사용하여 완성하시오.
- 조건은 다음과 같습니다.

## 조건
점수 | 등급
-----|----
90점 이상 | A
80점 이상 | B
70점 이상 | C
60점 이상 | D
60점 미만 | F

## 코드
```c
#include <stdio.h>

int main(){

    int num;
    char grade;

    printf("점수를 입력하세요: ");
    scanf("%d", &num);

    /*
    * 채워야 할 부분
    */
    
    printf("당신의 등급은 %c 입니다.", grade);

    return 0;
}
```

## 입력
N은 0보다 크거나 같고, 100보다 작거나 같다.

## 출력
출력형식과 같게 사용자가 점수를 입력했을 때, 등급을 알 수 있도록 출력한다.

### 예제 입력
```
75
```

### 예제 출력
```
점수를 입력하세요: 75
당신의 등급은 C 입니다.
```
