# 답
## 요구능력
- 함수에 대한 이해(반환타입 및 쓰임새)
- for문에 대한 이해
- 증감연산자에 대한 이해
- 포맷팅에 대한 이해

## 코드
```c
#include <stdio.h>

void gugudan(int num);

int main(){
    int num;
    printf("단을 입력하세요: ");
    scanf("%d", &num);
    gugudan(num);

    return 0;
}

void gugudan(int num){
    for(int i=num; i>=1; i--){
        printf("----------\n");
        for(int j=1; j < 10; j++){
            printf("%d * %d = %d\n", i, j, i*j);
        }
    }
}
```
